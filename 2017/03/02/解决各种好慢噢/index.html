结果由于国内网络不好，所以gem安装完全没反应。所以在国内，我们要想使用gem来安装相关的包的话，如果使用gem源地址，一般情况下都会安装失败，所以，我们需要将其替换为淘宝的Ruby包的网址就可以了。

然后就可以愉快的使用gem安装辣！
